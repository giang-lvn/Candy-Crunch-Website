## Project Overview

**Candy Crunch** is an educational e-commerce website project developed to simulate a **Direct-to-Consumer (D2C)** online shopping platform for **healthy, low-sugar and sugar-free candy products**.

The system is designed based on real-world e-commerce workflows, focusing on:

* User-friendly shopping experience
* Secure account management
* Order processing and tracking
* Admin management for products, orders, customers, promotions, and revenue

The project targets international consumer trends such as **healthy snacking, clean-label products, sugar-free candy, and cross-border e-commerce**, and applies modern web development practices following the **MVC architecture**.

## Main Features

### Customer Side

* User registration, login, password recovery

* Product detail with reviews and related items
* Wishlist management
* Shopping cart & checkout
* Voucher application
* Order tracking & history
* Account management (profile, address, payment info)

### Admin Side

* Secure admin authentication
* Dashboard with sales statistics
* Product, category, attribute & SKU management
* Order management (status updates, cancellations, returns)
* Customer management
* Voucher & promotion management
* Revenue tracking and reporting

## How to Run Our Project?

---------------------------------------------Start-----------------------------------------

1. Open **Visual Studio Code**
2. Open **Terminal** and clone the project:

```
git clone <your-github-repository-link>
```

3. Download and install **XAMPP**
4. Move the project source code into:

```
xampp/htdocs/
```

5. Open **XAMPP Control Panel**

   * Start **Apache**
   * Start **MySQL**

6. Open browser and go to:

```
http://localhost/phpmyadmin/
```

7. Create a new database:

```
Database name: candy_crunch
```

8. Import the database file located in the `database` folder

9. Run the project via browser:

**Admin**

```
http://localhost/Candy_Crunch_Website/admin/index.php

```

**User**

```
http://localhost/candy_crunch/web_user/login_user.php
```

> Note: URLs may vary depending on your local folder name.

---------------------------------------------End-----------------------------------------
